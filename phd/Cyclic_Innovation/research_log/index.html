Research Log & Backup Cheat Sheet

Backup Cheat Sheet (Pin This)

Golden rule: backup at the end of a research session, not the beginning.

Dry run (always first)

rsync -avhn --delete \
  ~/research/ \
  user@backuphost:/path/to/backup/research/
  • -a archive (permissions, timestamps)
  • -v verbose
  • -h human readable
  • -n dry run (no changes)
  • --delete mirror deletions

Real run (when dry run looks sane)

rsync -avh --delete \
  ~/research/ \
  user@backuphost:/path/to/backup/research/

With progress bar (optional)

rsync -avh --delete --progress \
  ~/research/ \
  user@backuphost:/path/to/backup/research/

One-off file copy (acceptable exception)

scp file.md user@backuphost:/path/to/backup/
